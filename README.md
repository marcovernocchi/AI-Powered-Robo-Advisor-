# AI-Powered-Robo-Advisor

A personalized investment advisory platform combining machine learning portfolio optimization, real-time market data, and AI-generated financial advice via an LLM. Built as a university project for Programming in Finance II (2026).

## Features

- **User authentication** — secure JWT-based registration and login
- **Risk profiling** — questionnaire that calculates a 1–10 risk score (conservative / moderate / aggressive)
- **Portfolio management** — track holdings with real-time prices and P&L from Yahoo Finance
- **Portfolio optimization** — mean-variance optimization via PyPortfolioOpt, tuned to your risk profile
- **AI advice** — personalized investment advice generated by Llama 3.3-70B (Groq, free tier)
- **Market explorer** — price history charts and company info for any ticker
- **Interactive dashboard** — Streamlit + Plotly dark-theme UI

## Tech Stack

| Component | Technology |
|---|---|
| Backend API | FastAPI + SQLAlchemy (SQLite) |
| ML / Finance | PyPortfolioOpt, yfinance, scikit-learn |

| Frontend | Streamlit + Plotly |
| Deployment | DigitalOcean VPS (Docker) |

## Installation

### Prerequisites
- Python 3.11+
- A [Groq API key](https://console.groq.com) (free)

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/paperellagialla11/AI-Powered-Robo-Advisor-Platform-.git
cd AI-Powered-Robo-Advisor-Platform-

# 2. Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate      # macOS/Linux
# .venv\Scripts\activate       # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Set up environment variables
cp .env.example .env
# Edit .env and add your GROQ_API_KEY and a SECRET_KEY

# 5. Start the backend (terminal 1)
uvicorn backend.main:app --reload

# 6. Start the frontend (terminal 2)
PYTHONPATH=. streamlit run frontend/app.py
```

Open [http://localhost:8501](http://localhost:8501) in your browser.

The FastAPI interactive docs are at [http://localhost:8000/docs](http://localhost:8000/docs).

## Usage

1. **Register** an account on the home page
2. **Complete the risk questionnaire** in the AI Advisor page
3. **Add holdings** in the Portfolio page (ticker, shares, average buy price)
4. **Run optimization** to see the suggested portfolio weights for your risk profile
5. **Generate AI advice** for personalized commentary on your portfolio
6. **Explore** any stock in the Market page

## API Documentation

The full interactive API is available at `/docs` when the backend is running.

| Method | Endpoint | Description |
|---|---|---|
| POST | `/auth/register` | Create account |
| POST | `/auth/login` | Get JWT token |
| GET | `/auth/me` | Current user info |
| POST | `/risk-profile` | Submit risk questionnaire |
| GET | `/portfolio/` | Get holdings with live prices |
| POST | `/portfolio/holdings` | Add a holding |
| DELETE | `/portfolio/holdings/{id}` | Remove a holding |
| GET | `/portfolio/optimize` | Run portfolio optimization |
| POST | `/advice/generate` | Generate AI advice |
| GET | `/advice/history` | Past advice |
| GET | `/market/history/{ticker}` | Price history |
| GET | `/market/info/{ticker}` | Company info |

## Deployment on Azure (Student Account)

1. Create a **B1s VM** (1 vCPU, 1 GB RAM) on Azure for Students ($100 free credits)
2. Install Docker on the VM
3. Clone the repo, create `.env`, then:
   ```bash
   docker compose up -d
   ```
4. Open ports 8000 (API) and 8501 (frontend) in the Azure network security group
